# Programação Estruturada Modular
## Professor: Carlos Menezes
## Curso: Análise e Desenvolvimento de Sistemas - FATEC Ipiranga
## Descrição
Este repositório contém os códigos e materiais utilizados nas aulas de Programação Estruturada Modular
ministradas pelo Prof. Carlos Menezes. O objetivo é fornecer exemplos práticos e exercícios para ajudar os alunos a entenderem os conceitos de programação estruturada e modularização.
## Estrutura do Repositório
- `aula-XX/`: Exercícios de cada aula
- `projetos/`: Projetos maiores
